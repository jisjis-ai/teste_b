Política de Privacidade

Versão Gratuita

Não coletamos nenhum dado pessoal dos usuários na versão gratuita. Os anúncios exibidos podem coletar dados de acordo com suas políticas.

Versão Premium

Coletamos informações dos usuários Premium, como email, nome e senha, armazenados com segurança no Firebase, para oferecer acesso exclusivo aos recursos Premium.

Anúncios

Na versão gratuita, os anúncios podem envolver a coleta de dados de acordo com as políticas dos anunciantes. Recomendamos revisar essas políticas. Na versão Premium, é possível desativar alguns anúncios, com um prazo de 24 horas para processamento.

Uso de Dados

Não utilizamos dados pessoais dos usuários da versão gratuita para nenhum propósito, além da exibição de anúncios. Na versão Premium, os dados são utilizados para personalizar a experiência do usuário.

Segurança

Todos os dados, especialmente os dos usuários Premium, são armazenados de forma segura no Firebase para proteção contra acesso não autorizado.

Alterações na Política de Privacidade

Reservamo-nos o direito de atualizar esta política. Alterações serão comunicadas aos usuários por notificações no aplicativo.

Ao usar nosso aplicativo, os usuários concordam com esta política de privacidade. Recomendamos a leitura completa antes de utilizar o aplicativo.
